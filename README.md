## English Version

GearHub is a single-vendor e-commerce web application designed for gamers to easily discover, compare, and purchase gaming peripherals.

Unlike a traditional online store, GearHub provides a built-in product comparison feature to help customers make informed purchasing decisions based on specifications and price.

### 📖 Project Overview

Choosing gaming peripherals can be confusing because there are many products with similar specifications and prices.

Most online stores only display product information, requiring customers to compare products manually.

GearHub aims to simplify this process by providing a clean shopping experience together with an easy-to-use product comparison system.

### 🎯 Objectives

- Develop a modern E-commerce website
- Practice Full Stack Web Development
- Apply Software Engineering concepts
- Design a relational database
- Implement shopping cart and ordering system
- Build a product comparison feature

## ภาษาไทย (Thai Version)

GearHub คือเว็บแอปพลิเคชันอีคอมเมิร์ซแบบผู้ขายรายเดียว (Single-vendor E-commerce) ที่ออกแบบมาสำหรับเกมเมอร์ เพื่อให้สามารถค้นหา เปรียบเทียบ และเลือกซื้ออุปกรณ์เกมมิ่งเกียร์ (Gaming Peripherals) ได้อย่างง่ายดาย

แตกต่างจากร้านค้าออนไลน์ทั่วไป GearHub มีฟีเจอร์สำหรับเปรียบเทียบสินค้าในตัว เพื่อช่วยให้ลูกค้าสามารถตัดสินใจซื้อได้อย่างถูกต้องและคุ้มค่าที่สุด โดยอ้างอิงจากข้อมูลสเปกและราคา

### 📖 ภาพรวมของโปรเจกต์ (Project Overview)

การเลือกซื้ออุปกรณ์เกมมิ่งเกียร์อาจสร้างความสับสนให้กับผู้ใช้ เนื่องจากมีสินค้าจำนวนมากที่มีสเปกและราคาที่ใกล้เคียงกัน

ร้านค้าออนไลน์ส่วนใหญ่จะแสดงเพียงแค่ข้อมูลของสินค้าแต่ละชิ้น ทำให้ลูกค้าต้องทำการเปรียบเทียบข้อมูลด้วยตัวเอง

GearHub มีเป้าหมายที่จะทำให้กระบวนการนี้ง่ายขึ้น โดยการมอบประสบการณ์การช็อปปิ้งที่เรียบง่าย สะอาดตา ควบคู่ไปกับระบบเปรียบเทียบสินค้าที่ใช้งานง่าย
